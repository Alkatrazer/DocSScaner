# Notice of modifications

DocSScaner is a modified version of [FairScan 2.1.0](https://github.com/pynicolas/FairScan), originally developed by Nicolas Pensart and FairScan contributors.

The changes for DocSScaner 1.0 were completed and published on **July 20, 2026** by **Alkatrazer**. The source corresponding to the RuStore release is marked by the [`v1.0-rustore`](https://github.com/Alkatrazer/DocSScaner/tree/v1.0-rustore) tag.

The main modifications are:

- renamed the application to DocSScaner and changed its visual and store identity;
- changed the Android namespace and application ID to `ru.alkatrazer.docscaner`;
- added RuStore metadata, Russian release information, and a Russian privacy policy;
- added a private release-signing workflow and local scripts for producing the RuStore APK;
- tightened release and Android data-protection settings for the repackaged application;
- updated project documentation and contact information for the DocSScaner distribution.

The original FairScan copyright notices remain applicable. DocSScaner is distributed under the same [GNU General Public License version 3](LICENSE).
