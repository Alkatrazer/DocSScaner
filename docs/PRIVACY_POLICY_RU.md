# Политика конфиденциальности DocSScaner

Дата вступления в силу: 20 июля 2026 года.

DocSScaner предназначен для сканирования документов и создания PDF-файлов. Обработка изображений, распознавание текста и формирование PDF выполняются локально на устройстве пользователя.

## Какие данные обрабатывает приложение

- Изображения с камеры и импортированные пользователем изображения — только для создания сканов.
- Распознанный текст — только для добавления текстового слоя в PDF.
- Созданные PDF-файлы — сохраняются или передаются только по явному действию пользователя.
- Технические журналы — хранятся локально и отправляются разработчику только если пользователь сам выберет отправку через установленное почтовое приложение.

DocSScaner не создаёт учётные записи, не использует рекламные идентификаторы, аналитику или трекеры, не продаёт и не передаёт отсканированные документы третьим лицам.

## Доступ к интернету

Интернет используется только по запросу пользователя для загрузки выбранных языковых файлов OCR из официального репозитория Tesseract на `raw.githubusercontent.com`. При таком обращении оператор сетевого ресурса может получить стандартные технические сведения соединения, например IP-адрес, в соответствии со своей политикой конфиденциальности. Изображения документов и распознанный текст при этом не отправляются.

## Разрешения Android

- Камера — для съёмки документов.
- Интернет — для загрузки дополнительных языков OCR.
- Хранилище на Android 9 и более ранних версиях — для сохранения созданного PDF в выбранное пользователем место.

## Хранение и удаление

Рабочие изображения, настройки, OCR-файлы и журналы находятся на устройстве пользователя. Приложение запрещает их облачное резервное копирование средствами Android. Рабочие изображения текущего документа удаляются при завершении работы с ним. Пользователь может удалить данные приложения средствами Android или удалить приложение.

## Дети

Приложение не содержит контента с возрастными ограничениями и не предназначено для сбора персональных данных детей.

## Изменения политики

Актуальная версия политики публикуется по тому же адресу, который указан в карточке приложения RuStore. При существенных изменениях новая дата вступления в силу будет указана в начале документа.
